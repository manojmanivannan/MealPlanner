MealPlanner

A simple way to plan your meals

About the Project

MealPlanner is an open-source tool designed to help organize your daily or weekly meals. It allows users to plan recipes, track ingredients, and save time.

Getting Started

  1. Clone the repository: git clone https://github.com/manojmanivannan/MealPlanner.git
  2. Install dependencies (see project README)
  3. Run locally and start planning meals
